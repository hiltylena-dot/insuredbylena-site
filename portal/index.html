                    Missing Link
                    Quick manual

                    1. Pick a post: Use Quick Pick or the post list.

                    2. Edit the plan here: Update hook, caption, CTA, reel notes, and Canva layout notes.

                    3. Push design changes to Canva: Use Copy Canva Handoff, make the visual/text changes in Canva, and export the final asset there.

                    4. Paste back the exported asset: Put the final public file into Final Asset URL.

                    5. Approve + schedule: Click Approve, then set Schedule At and click Schedule.

                    6. Publish: Click Run Publish. It sends all due approved or scheduled posts.

                    If nothing happens: The post is usually still draft, scheduled for later, or missing a real public asset URL.
